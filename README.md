# Alex's HA Add-Ons

Custom Home Assistant add-on repository.

## Installation

1. In Home Assistant, go to **Settings → Add-ons → Add-on Store**
2. Click the **⋮** menu (top right) → **Repositories**
3. Add: `https://github.com/alexlmiller/ha-addons`
4. The add-ons below will appear in the store

## Add-ons

| Add-on | Description |
|--------|-------------|
| [ScanSnap iX500](./scansnap/) | One-button OCR scanning to modular storage backends (Nextcloud first) |
